# Blog Content Sprint — AI-SEO Article Production

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish 50+ SEO-optimized blog articles on incrediblehospitalityco.com/blog/ to own "micro resort" and adjacent hospitality investing queries in AI tools and Google search.

**Architecture:** Static HTML articles using a shared template, each in its own `/blog/{slug}/index.html` directory (clean URLs on GitHub Pages). Each article gets Article + FAQPage JSON-LD schema, OG tags, author byline with credentials, and internal links to mastermind page and challenge page. A blog index page lists all articles. Sitemap is updated after each batch.

**Tech Stack:** Static HTML/CSS on GitHub Pages. Same design system as main site (dark green + gold). No JS framework. Facebook Pixel on every page.
